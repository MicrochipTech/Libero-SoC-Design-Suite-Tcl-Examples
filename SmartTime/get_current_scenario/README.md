# get_current_scenario

## Description 

This Tcl command returns the name of the current timing scenario.

```
get_current_scenario
```

## Arguments 

|Parameter|Type|Description|
|---------|----|-----------|
|None|None|None|

|Return Type|Description|
|-----------|-----------|
|string|Name of the current timing scenario.|
